Getting Started

Follow these essential steps to run your debate event from registration to final results.

Step 1: Register Participants

  1. Click "Register Participant" from the dashboard
  2. Fill in: Name, College, Email, Phone, Topic, Experience Level
  3. Click "Register Participant" to save
  4. Repeat for all participants
Note: Each participant needs a unique email address.

Step 2: Judge Login

  1. Click "Judge Login" from the dashboard
  2. Enter: Username: judge | Password: admin123
  3. Click "Login" to access judge features
Security: Use logout after finishing judge operations to protect access.

Step 3: Manage Participants

  1. Click "Participant Manager" (requires judge login)
  2. Assign debate sides: "For" or "Against"
  3. Optional: Filter by topic or side
  4. Optional: Export lineup as CSV

Step 4: Schedule Debates

  1. Click "Schedule Slots" (requires judge login)
  2. For each participant, set: Date, Time, Venue
  3. Click "Save Schedule"

Step 5: Score Participants

  1. Login as judge (if not already logged in)
  2. For each participant, assign scores:
    • Content: 0-30 points (argument quality)
    • Delivery: 0-30 points (presentation)
    • Rebuttal: 0-40 points (counter-arguments)
  3. Total is calculated automatically (max 100)
  4. Click "Update Score" to save

Step 6: View Results

  1. Click "Results" from the dashboard
  2. View top performers with medal rankings
  3. Check winning side analysis
  4. Review complete rankings table

Step 7: Generate Certificates

  1. Go to Results page
  2. Click "Generate Certificate" for participant
  3. Click "Download PDF" button
  4. Certificate saves with participant name and scores
Tip: Use "Generate All Certificates" to create and download PDFs for all participants at once.

Need More Help?

Judge Features: Participant Manager, Schedule Slots, and scoring pages require login.
Public Features: Registration and Results are accessible to everyone.
Logout: Click the logout button when finished to secure judge access.

Common Issues & Solutions

Email already exists during registration

Each participant must have a unique email. Check existing entries or use a different email address.

Scores not updating

Enter valid values only: Content 0-30, Delivery 0-30, Rebuttal 0-40, then click "Update Score".

Certificate download not working

Allow browser downloads and ensure participant data is complete before generating certificates.

Data disappeared after closing browser

Data is stored in browser localStorage. Avoid clearing browser data and export logs/lineup regularly.

Start fresh for a new event

Login as judge, open Participant Manager, and use "Clear All Data" after completing backups.

Multiple judges using the system

Judges share the same data and credentials, so coordinate updates to avoid scoring conflicts.

Best Practices

  • Register all participants before the event begins
  • Assign sides and schedules in advance
  • Use a consistent scoring rubric
  • Score immediately after each debate
  • Export lineup and logs as backup
  • Generate certificates after finalizing results
